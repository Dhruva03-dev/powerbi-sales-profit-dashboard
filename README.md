# 📊 Sales & Profit Analysis Dashboard (Power BI)

## 🧭 Overview

This project presents a multi-page interactive Power BI dashboard built using a retail dataset. The goal is to analyze sales performance, profitability, and customer behavior, and transform raw data into meaningful business insights.

The report follows a structured flow, allowing users to move from a high-level overview to detailed exploration.

---

## 📑 Report Structure

### 🔹 Overview

Provides a snapshot of overall performance with key metrics, sales and profit trends, and regional distribution.

### 🔹 Sales Analysis

Explores revenue patterns across categories, sub-categories, and products, along with time-based trends.

### 🔹 Profit Analysis

Focuses on profitability, highlighting loss-making areas and comparing profit against sales to identify inefficiencies.

### 🔹 Customer Segment Analysis

Analyzes customer segments based on sales, order frequency, and average order value across regions.

### 🔹 Detailed Insights

Offers a drill-down view using a matrix with conditional formatting to identify high and low-performing products.

---

## 📊 Key Insights

* Sales growth does not always translate into profit growth
* Certain sub-categories generate high sales but low or negative profit
* Consumer segment drives most orders, while other segments may offer higher value per order
* Profitability varies significantly across regions and product categories
